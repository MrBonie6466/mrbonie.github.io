Play

Move

Move the chess pieces the same way as in your normal chess. If you do not the moves the link provided below should help

Link

Take/Check

Once the piece is taken or check appeared, digital chess will automatically display it on.

Checkmate

Once the checkmate is done, the game will end.

Promotion

Once the pawn reaches the opposite end, the chance of promotion would appear. Promotion is done by pressing the button in specific cycle (Queen, Rook, Bishop, Knight). IMPORTANT: If no piece is selected, it will be set to Queen by default

Reset

If you want to reset the game, you would need to rearrange the pieces to original positions. Then, press reset button to start a new game.
